# 航运教学演示平台场景映射表

1. `S1` 集装箱码头泊位分配与岸桥调度
2. `S2` 码头堆场重定位与场桥协同
3. `S3` 海运库存与航线联合决策

## 5. 后续深化接口建议

### `S1` 后续深化

- 联合优化高级模式
- 岸桥不可穿越与安全距离约束
- 最大完工时间/等待时间双目标切换

### `S2` 后续深化

- 箱位分配与车辆调度联动
- `makespan` 与等待时间并行指标
- 车辆路径与堆场网格的协同动画

### `S3` 后续深化

- 滚动优化与启发式策略对比
- 多船版本与港口服务时间
- 缺货罚金与持有成本权重实验

### `S4` 后续深化

- 卸船机、堆场、水平运输资源协同
- 时间/成本/能耗三类指标
- 干散货全流程瓶颈识别
