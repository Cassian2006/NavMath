📘 数学建模与算法原理

目标函数

min C = Cfixed + Cvariable + Cpenalty

  • Cfixed：固定成本（集卡购置、维护）
  • Cvariable：可变成本（燃油、人工）
  • Cpenalty：惩罚成本（延误、拥堵）

约束条件

  • 集卡数量：0 ≤ T ≤ Tmax
  • 岸桥作业时间：tbridge ≤ tmax
  • 堆场容量：Qyard ≤ Qmax
  • 路径长度：L ≤ Lmax

遗传蚁群算法 (GA-ACO)

遗传算法 (GA)
  • 种群初始化：随机生成路径方案
  • 选择操作：轮盘赌选择适应度高的个体
  • 交叉操作：两点交叉产生新路径
  • 变异操作：随机交换路径节点
蚁群算法 (ACO)
  • 信息素更新：τij = (1-ρ)τij + Δτij
  • 状态转移：pij = [τijα × ηijβ] / Σ
  • 启发函数：ηij = 1 / dij（距离倒数）
